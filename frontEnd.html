Front End & Web

O Front End na Web, é um dos grandes responsáveis pela personalidade de uma página na internet, se trata de todo o visual, aquilo que o usuário verá.

Atualmente existem diversos estudos sobre a experiência e a interface do usuário (UX/UI). Existem profissionais altamente capacitados para projetar uma interface que agrade os usuários de maneira geral.

Além do design, também existem boas práticas que são encontradas em todo bom site, tais convenções garantem organização no código, bom desempenho e maior visibilidade para mecanismos de busca.
